SimpleX Chat is an open-source messaging platform built around a single radical principle: **no user identifiers of any kind**. No phone numbers, no email addresses, no usernames, no random IDs — nothing that could link your identity to your messages. All communication is end-to-end encrypted using the double ratchet algorithm with an extra NaCl encryption layer, and the network is fully decentralized so you can run your own servers.

## Get started in 3 steps

<Steps>
  <Step title="Download SimpleX Chat">
    Install the app on [iOS](https://apps.apple.com/us/app/simplex-chat/id1605771084), [Android](https://play.google.com/store/apps/details?id=chat.simplex.app), or [desktop](https://github.com/simplex-chat/simplex-chat/releases/latest). No account, phone number, or email required — just create a local profile name.
  </Step>
  <Step title="Make your first connection">
    Share a one-time invitation link or QR code with a contact. SimpleX connections are formed by exchanging one-way message queue addresses — neither party needs a global identifier.
  </Step>
  <Step title="Start messaging privately">
    All messages are end-to-end encrypted with double ratchet + NaCl. Enable Incognito mode for per-contact pseudonyms, or set up disappearing messages for ephemeral chats.
  </Step>
</Steps>
